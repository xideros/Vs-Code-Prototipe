# 🎮 Прототип 5 — твой помощник для автоматизации игр!

Добро пожаловать! Этот проект — не просто набор скриптов, а целая лаборатория для экспериментов с автоматизацией, распознаванием текста, обработкой аудио и многопоточностью в Python.

---

## 🚀 Быстрый старт

1. Скачайте проект:
   ```
   git clone <URL-репозитория>
   ```
2. Перейдите в папку:
   ```
   cd "Прототип 5"
   ```
3. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
4. Запустите:
   ```
   python main.py
   ```
