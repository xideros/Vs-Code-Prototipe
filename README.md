Цель проекта

Ассистент помогающий получать лучший игровой опыт по локализации. Это интеллектуальный ассистент для геймеров,
который автоматически распознаёт субтитры на экране игры, переводит их на русский язык и озвучивает с помощью синтеза речи.
Программа работает в реальном времени: выделяет область экрана, захватывает текст с помощью OCR, определяет язык, при необходимости переводит, 
синтезирует аудио и воспроизводит его. Всё происходит автоматически и параллельно, чтобы пользователь мог слышать перевод субтитров сразу во время игрового процесса, 
без задержек и ручных действий.

Установка
Еще не совсем программа, есть только куча файлов и папок.
